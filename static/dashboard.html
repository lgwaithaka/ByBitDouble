Strategy Parameters

Risk Management — What Each Setting Does

• epoch_max_dd_pct: Halts all trading if balance drops this % from epoch start (default 35%)

• daily_max_dd_pct: Halts if today's drawdown exceeds this % (default 25%). Resets midnight UTC.

• vol_scan_n: How many volatile symbols to scan per cycle (auto-sized by balance)

• scan_interval_s: Seconds between full scan cycles (default 45s)

• Signal gates: 4H macro → 1H structure → 15m/5m entry. ALL must align before trading.

• SL = 2.0× ATR: Stop placed behind market structure, not at random level.

• TP = 3.5× ATR: Minimum 1:1.75 R:R on every trade.
